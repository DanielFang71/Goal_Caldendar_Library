Getting started

  • Add this scripts to <head> in your html file: 
    •                 <link type="text/css" href="../pub/css/calendar.css" rel="stylesheet">
  • Add these scripts to <body> in your html file: 
    •                 <div id="calendarContainer"></div>
                <div id="organizerContainer"></div>
                Where calendarContainer and organizerContainer are the id of your container in html page.
                <script src="../pub/js/calendar.js"></script>
                <script src="../pub/js/yourexample.js"></script>

    • API Section

  • Calendar object properties:

    • Property Type Description
    id String The id of your calendar container in html.
    size String The size of the calendar. Options are small | medium | large.
    label setting Array of String The label settings for the calendar. First entry indicates the leftmost day of calendar labels, and second entry is the maximum length of the calendar labels.
    colors Array of String The color settings for the calendar. First entry is the primary color for calendar, second is the primary dark color, third is the text color, and fourth is the text dark color.
    options Object The option settings for the calendar. (language and placeholder) Inside object optional keys: days, months and placeholder. Each key refer to an array of String to represent day labels and month labels. Placeholder refers to a placeholder html code.
    goalsObj Empty array The default goals collections, later input goal complete objects will be stored here.

  • Calendar object's methods:

    • Methods Description
    changeColor Input a new array of color strings to change the calendar's color setting.
    changeSize Input a new size string to change the calendar's size.
    changeLableSetting Input a new array of label strings to change the calendar's label Setting.
    createGoal Input goal current amount, goal target amount, and goal detail text to create a goal info object. Return the goal info object.
    addGoalToObjs Input a Date object and goal info object to create complete goal object. Return the complete goal object.
    addData Input the complete goal object and add it to the calendars goal collections. Return the calendar's goal collections.

  • Organizer object properties:

    • Property Type Description
    id String The id of your organizer container in html.
    calendar Object The calendar object instance you want to connect with.
    data Object The goal data you want to add to your calendar. Use Calendar's createGoal, addGoalToObjs, addData methods to create the goal data.

  • To create goals data for adding to Organizer object always use the combination of calendar's methods createGoal,addGoalToObjs and addData, see the example above and also see example codes.
